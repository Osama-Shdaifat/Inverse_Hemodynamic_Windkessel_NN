# Inverse_Hemodynamic_Windkessel_NN
Official implementation of the paper: Using Physics-Guided Neural Networks to Resolve the Inverse Hemodynamic Problem of the 4-element Windkessel Model and Cardiac Diagnosis


Use the 256_Physics_Flow_Generation.py file to generate the heart flow data.
Use the 256_Physics_PPG_Generation.py file to generate the arterial blood pressure data using the windkessel model.
Use the 256_Physics_Conv_Transformer.py file to train the neural network. 
Use the 256_mgh_Patient_Testing_Conv_Specialists_Diagnoses_NonNormalized.py to test the model.
